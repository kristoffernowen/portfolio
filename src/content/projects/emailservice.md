## Overview

A lightweight microservice built with .NET for sending transactional emails. Integrates with SendGrid for reliable email delivery and supports templated messages.

## Features

- Templated email support
- Retry logic with exponential backoff
- Health check endpoint
- Docker-ready deployment

## Tech Stack

- .NET 8
- SendGrid
- Docker
