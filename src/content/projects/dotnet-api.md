## Overview

A production-ready REST API built with ASP.NET Core. Implements JWT-based authentication, role-based authorization, and uses Entity Framework Core for data access.

## Features

- JWT authentication & refresh tokens
- Role-based authorization
- Swagger/OpenAPI documentation
- Entity Framework Core with migrations
- Repository pattern

## Tech Stack

- .NET 8 / ASP.NET Core
- Entity Framework Core
- SQL Server
- Swagger / OpenAPI
