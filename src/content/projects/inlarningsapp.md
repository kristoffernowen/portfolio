## Overview

Projektet började som ett sätt att träna på flexibla formulär, Clean Architecture, CQRS och TDD genom att bygga ett system för fakta-inlärning. API:et låter användare skapa modeller som används för att generera övningsobjekt med frågor och svar.

React-klienten hanterar skapande och genomförande av övningar, medan backend returnerar frågor, tar emot svar och skickar tillbaka underlag för rättning och feedback.

## Features

- Skapa och hantera egna övningar för fakta-inlärning
- API med Clean Architecture och CQRS
- Dynamiska formulärflöden i React-klient
- Genomförande av övning med direkt återkoppling
- Driftad lösning i Azure med CI/CD via GitHub
